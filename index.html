COMPÉTENCES

HTML5

L'HyperText Markup Language désigne un type de langage informatique descriptif. Il s'agit plus précisément d'un format de données utilisé dans l'univers d'Internet pour la mise en forme des pages Web.

Plus d'infos

CSS3

CSS est l'acronyme de « Cascading Style Sheets » ce qui signifie « feuille de style en cascade ». Le CSS correspond à un langage informatique permettant de mettre en forme (couleurs, taille...) des pages web (HTML ou XML).

Plus d'infos

Javascript

JavaScript désigne un langage de développement informatique. On le retrouve principalement dans les pages Internet. Il permet, entre autres, d'introduire sur une page web HTML des petites animations ou des effets.

Référencement SEO

Le marketing SEO ou le référencement internet est une stratégie de marketing digital qui se concentre sur la présence de votre site Web dans les résultats de recherche sur les moteurs de recherche ou annuaires en ligne.
